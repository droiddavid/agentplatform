Foundation

- repository skeleton
- frontend shell
- backend shell
- config and environment support
- database connection
- Flyway
- auth and user foundations

## Phase 2
Onboarding and templates

- onboarding flows
- template seed data
- template APIs
- template browser UI

## Phase 3
Agent domain

- agent tables
- agent CRUD
- wizard creation
- natural language creation
- template-based creation

## Phase 4
Tasks and runs

- tasks
- runs
- run state machine
- event persistence
- run detail UI

Messaging and collaboration

- agent messages
- shared board
- shared context
- workspace
- graph projection

## Phase 6
Runtime execution engine

- model provider abstraction
- prompt assembly
- turn cycle
- approval gate evaluation
- spawn logic
- completion logic

## Phase 7
Connections and tools

- connection framework
- email
- calendar
- web search and browse
- document generation
- e-commerce adapter
- custom API framework

## Phase 8
Approvals, memory, and privacy

- approval inbox
- remembered approval rules
- memory dashboards
- export
- deletion
- consent and audit

## Phase 9
Billing and usage

- plan model
- entitlements
- usage tracking
- subscription and invoices

## Phase 10
Notifications, PWA polish, final validation

- notifications
- streaming
- help and UX copy
- responsive pass
- accessibility pass
- acceptance test pass
