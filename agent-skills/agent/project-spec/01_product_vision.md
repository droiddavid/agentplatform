# Product Vision

## Product

A consumer-ready runtime AI agent platform.

## Primary Users

- General consumers
- Solo entrepreneurs
- Small businesses
- Creators
- Families and households

## Main User Outcomes

The platform must let non-technical users:

- create one or more AI agents
- define their responsibilities in plain language
- connect tools and services
- assign tasks
- review multi-agent collaboration
- approve external actions
- allow remembered approvals if desired
- manage privacy, memory, and data export

## Product Principles

1. Plain-language user experience
2. Human control over external actions
3. Clear explanation of what the system is doing
4. Runtime flexibility over hard-coded roles
5. Auditability and trustworthiness

## Version 1 Experience

The user should be able to:

- create an account
- complete onboarding
- create an agent by wizard, natural language, or template
- create a task
- launch a run
- observe agents collaborating
- approve or deny external actions
- review generated outputs
- manage privacy and billing

## Product Success Criteria

A user should be able to create an agent in under ten minutes and complete
a meaningful task in under fifteen minutes, while still understanding what
the system is doing and why.
