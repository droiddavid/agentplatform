SEED DATA AND CURATED TEMPLATE PACK
Runtime AI Agent Platform
Version 1

==================================================
1. PURPOSE
==================================================

This pack defines the initial seed data required for a usable product on first boot.
It includes:
- subscription plans
- feature entitlements
- template categories
- curated agent templates
- tool definitions
- retention policies

The objective is to ensure the product is consumer-ready with meaningful defaults.

8. RECOMMENDED FIRST-BOOT EXPERIENCE DATA
==================================================

The product should feel ready immediately after startup.
Therefore, the seed data must allow:

1. Template browser to render meaningful cards
2. Plan page to show actual plans
3. Agent creation wizard to suggest curated defaults
4. Tool permission UI to display human-readable tool labels
5. Privacy settings to reference real retention policy concepts

==================================================
9. IMPLEMENTATION NOTES
==================================================

1. Seed data should be inserted by Flyway migrations after all schema creation migrations.
2. Template spec JSON should be valid and machine-consumable.
3. Tool definitions should be stable and code-driven.
4. Seed content should use clear plain-language descriptions for non-technical users.
5. Future marketplace data should not be mixed into v1 seed data.
