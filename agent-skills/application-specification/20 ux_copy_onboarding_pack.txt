
UX COPY AND ONBOARDING TEXT PACK
Runtime AI Agent Platform
Version 1

==================================================
1. PURPOSE
==================================================
This pack provides all user-facing copy used in onboarding, dashboards,
agent creation, approvals, and help guidance. Language is written for
non‑technical users and must remain simple, supportive, and explanatory.

Tone guidelines:
- Friendly
- Clear
- Non‑technical
- Trust‑building
- Action‑oriented
